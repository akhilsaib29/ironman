# IronHand 🤖
Control your computer with hand gestures — Iron Man JARVIS style!

## Install
```
pip install -r requirements.txt
python -m ensurepip --upgrade
```

## Run
```
python main.py
```

## Gestures
| Gesture | Action |
|---|---|
| ✋ Open Palm | Volume UP |
| ✊ Closed Fist | Volume DOWN |
| ☝️ Index finger up | Scroll UP |
| ✌️ Peace sign | Scroll DOWN |
| 👍 Thumbs up | Play / Pause |
| 🤌 Pinch | Left Click |

## Notes
- Press **Q** to quit
- Works on Windows and Mac
- Make sure your webcam is connected
